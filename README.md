# EaseIn

A lightweight static website for practicing everyday interactions through
step-by-step flash-card decks.

## Run Locally

No package installation or build process is required. Open `index.html`
directly in a browser. The same directory can also be hosted by any static
file server when deploying or testing through `localhost`.

## Technology

- Semantic HTML
- Custom responsive CSS
- Vanilla JavaScript
- Locally curated activity data

## Included MVP Features

- Eight activity decks in five categories
- Activity search and category filtering
- Flash-card viewer with previous, next, and restart actions
- Likely-response prompts and practical phrases
- Keyboard support for filters, controls, `Escape`, and arrow-key card navigation
- Responsive layouts for mobile and desktop
